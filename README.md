# 早稲田法・日本史トレーナー（MVP v0.1）

早稲田大学法学部の日本史対策アプリ。傾向を可視化し、**ゼロから作ったオリジナル問題**で反復演習し、学習ログを残す自習ツール。

> **著作権方針（最重要）**：過去問の問題文・選択肢・解説は一切複製しない。テーマの傾向データと、独立に創作したオリジナル問題のみを扱う。実在問題の言い換え再現も禁止。問題本体は正規の赤本へ誘導する。

---

## ファイル構成

```
waseda-nihonshi-trainer/
├── index.html    ← アプリ本体（HTML+CSS+JS 全部入り）
├── data.js       ← 初期データ（傾向・問題・ログ種・リンク）※編集はここ
├── README.md     ← このファイル
└── render.yaml   ← Render 静的サイト設定
```

機能はすべて `index.html` に入っている。**中身（問題・傾向・リンク）を差し替えるときは `data.js` だけ編集**すればよい。

## ローカルで動かす

`data.js` を `<script src>` で読むため、**ファイルを直接ダブルクリックするだけでも動く**が、確実なのは簡易サーバー経由。

### 方法A：ダブルクリック
`index.html` をブラウザで開くだけ。（一部ブラウザのファイル制限が出たら方法Bへ）

### 方法B：簡易サーバー（推奨）
このフォルダで以下のいずれかを実行し、表示されたURLを開く。

```bash
# Python がある場合
python -m http.server 8000
#  → http://localhost:8000

# Node がある場合
npx serve .
```

> **注意**：学習ログは `localStorage` に保存される。Claude.ai のプレビュー内では localStorage が動かないため、**実データ保存の確認は必ずブラウザに配置してから**行うこと。

## データの差し替え方（`data.js`）

| 定数 | 中身 | 編集のコツ |
|---|---|---|
| `FIELDS` | 分野マスタ | ヒートマップ縦軸・棒グラフ・演習フィルタに連動 |
| `FORMATS` | 出題形式 | 正誤／空欄補充／論述／一問一答 |
| `YEARS` | ヒートマップの年度枠 | 直近10年。増やせば列が増える |
| `TRENDS` | 年度×テーマの傾向 | 1行＝「その年にこの分野がこの形式で出た」。順次追記 |
| `QUESTIONS` | 問題プール | `type` は `truefalse`／`short`／`essay`。下記参照 |
| `LOGS_SEED` | ログの種 | 初回だけ流し込むダミー。`[]` にすれば種なし |
| `LINKS` | 赤本等リンク | `url` を各自設定。空欄は「未設定」表示 |

### 問題の書き方

```js
// 正誤問題
{ id:"q001", field:"近代", difficulty:2, type:"truefalse",
  question:"問題文", choices:["正しい","誤っている"], answer:0,
  explanation:"解説" }

// 一問一答（short）… answer に正解文字列。表記ゆれは自己判定ボタンで救済
{ id:"q004", field:"中世", difficulty:2, type:"short",
  question:"問題文", answer:"北条泰時", explanation:"解説" }

// 記述（essay）… 自己採点式。模範解答を表示し、本人が○×を選ぶ
{ id:"q008", field:"社会経済", difficulty:4, type:"essay",
  question:"問題文", modelAnswer:"模範解答", explanation:"採点の観点" }
```

> **登録前に必ず史実確認**すること（AI生成問題は事実誤りが混じるリスクがある）。

## 使い方（4画面）

1. **傾向分析** … 年度×テーマのヒートマップ／分野別頻度の棒グラフ／出題形式の内訳。印刷ボタンでA4出力。
2. **演習** … 分野・難易度・形式・出題順で絞って出題。誤答は自動で復習リストへ。記述は模範解答→自己採点。
3. **学習ログ** … 連続学習日数・累計・正答率・日別推移・分野別正答率。CSV書き出し／読み込み。復習リスト印刷。
4. **赤本リンク** … 年度別リンク＋弱点分野に対応する年度を推奨表示。
